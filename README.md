# multiarc_criticality

This repository contains the data and code accompanying a paper on arc criticality analysis in supply chain networks. It supports the identification and ranking of critical arcs based on failure scenarios and their impact on network performance.

## Workflow

The analysis is structured as a sequential pipeline across the following scripts:

| Script | Description |
|---|---|
| `a_preproc.py` | Reads raw Java simulation exports (`data/raw/`), builds SQLite databases, and generates processed CSVs with arc and node information including coordinates. |
| `b_exploracion.py` | Exploratory analysis of arc and node attributes, failure probabilities, and KPI distributions from the database. |
| `c_nodos_inter_reg.py` | Identifies arc interaction patterns using decision trees and regression; outputs subset results per scenario. |
| `d_georefer_analysis.py` | Georeferenced visualization of the network using node coordinates; generates interactive maps with Folium. |
| `e_graph_analysis.py` | Graph-theoretic analysis of the supply chain network using NetworkX (centrality, connectivity, cut sets). |
| `f_RL_arc_importance.py` | Estimates arc importance via linear regression models; outputs arc importance rankings. |

## Data structure

```
data/
├── raw/        # Original .txt exports from Java simulation
├── processed/  # CSVs generated by Python (arc info, node index, coordinates)
├── viz/        # Excel files for inspection
└── db/         # SQLite database(s)
```

Results are stored in `resultados/`.
